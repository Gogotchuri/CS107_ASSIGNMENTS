This project is Assignment 1 of stanford course Programming Paradigms CS107;

This solution generates random sequence, based on specific grammar file,
for more information check "Assignment-1-RSG.pdf."

To build the project, run ' make ' command from console.
To run program ->  ./rsg <grammar file path>
example: ./rsg data/bionic.g
